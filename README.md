# BDMO National Camp Problem Sets 2026

This repository contains mathematical problem sets collected from the Bangladesh Mathematical Olympiad (BDMO) National Camp classes of 2026.

## Topics

The repository may include problems from:

- Number Theory
- Algebra
- Combinatorics
- Geometry

## Purpose

The goal of this repository is to help olympiad students practice and prepare for higher-level mathematical problem solving. 
These materials are intended for educational and non-commercial use only.
